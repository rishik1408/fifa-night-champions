FIFA NIGHT CHAMPIONS

Project Title: FIFA NIGHT CHAMPIONS

Subtitle: Football Tournament Management System

Overview: FIFA NIGHT CHAMPIONS is a web-based tournament management system designed to bring the prestige and atmosphere of the UEFA Champions League to local gaming nights. While the name evokes the excitement of a "FIFA Night," the core application is a robust, broadcast-style tool that allows organizers to set up, track, and visualize football tournaments with professional-grade aesthetics. It combines the utility of a bracket manager with the visual flair of top-tier European football coverage.

Key Features:

-UEFA Champions League Themed Experience:

-Visual Identity: Designed with a "Midnight" aesthetic featuring Royal Blue and Electric Cyan accents, mirroring the iconic look of European nights.

-Immersive Media: Supports high-quality, loopable video backgrounds and transparent PNG team/tournament logos to create a premium, TV-broadcast feel.

Versatile Tournament Modes:

-League Mode: Automated round-robin table generation with real-time tracking of Points, Goal Difference, and Form.

-Knockout Mode: A classic elimination bracket system where winners advance instantly.

-Group Stage: Manages dual groups (A & B) feeding into a final knockout phase.

-Playoffs (Qualifier Format): A specialized bracket visualizer (Qualifier 1, Eliminator, Qualifier 2, Final) for high-stakes finishes.

Interactive Dashboard:

-Match Centre: A streamlined interface for rapid score entry.

-Live Visualizers: SVG-based flowcharts and standings tables that update instantly without page reloads.

-User-Friendly Controls:

-Soft Reset: Reset tournament data instantly while keeping the immersive background video playing smoothly.

-Flexible Inputs: Case-insensitive text inputs that automatically format for a clean, professional look.

Technical Stack:

-Core: HTML5, CSS3, Vanilla JavaScript (ES6+).

-Media: Optimized HTML5 Video (WebM/MP4 support) and transparent image blending.
